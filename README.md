# Scuba dive MOD calculator
A simple MOD Calculator for nitrox mixes. This Calculation is assuming a Ppo2 of 1.4 ata. 
Enter your oxygen percentage as a whole number in the box. Ex: 32% will be entered as 32 
Happy Diving, Enjoy!
